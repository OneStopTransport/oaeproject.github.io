<p style="box-sizing: border-box; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;">The Apereo Open Academic Environment (OAE) project team is extremely proud to announce the next major release of the Apereo Open Academic Environment; OAE Falcon, or OAE 7.</p><p style="box-sizing: border-box; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;">OAE Falcon comes with a complete overhaul of the OAE email notification experience, as well as a more complete set of utilities for user and system management.</p>
<!--more-->
<h2>Changelog</h2><h3>Email Preferences</h3><p style="box-sizing: border-box; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;">A frequent request from OAE users is to provide greater control over how often they receive email. OAE Falcon provides the ability for users to specify how often they wish to receive email:</p><ul class="task-list" style="box-sizing: border-box; padding: 0px 0px 0px 30px; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;"><li style="box-sizing: border-box;"><em style="box-sizing: border-box;">Immediately</em>&nbsp;- When selected, users will receive an e-mail immediately after an important activity has taken place. When multiple important activities happen in quick succession, they are aggregated into a single e-mail</li><li style="box-sizing: border-box;"><em style="box-sizing: border-box;">Daily</em>&nbsp;- When selected, users will receive an e-mail once per day containing all the notifications the user received in the last 24 hours</li><li style="box-sizing: border-box;"><em style="box-sizing: border-box;">Weekly</em>&nbsp;- When selected, users will receive an e-mail once per week containing all the notifications the user received in the last 7 days</li></ul><p style="box-sizing: border-box; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;">Regardless of the email preference, users can still access their notifications from the OAE user interface in real-time as they occur.</p><p style="box-sizing: border-box; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;"><img style="display: block; margin-left: auto; margin-right: auto;" src="http://i.imgur.com/2TYQjyk.png" alt="Account Settings Modal Screenshot" width="437" height="379" /></p><h3>&nbsp;</h3><h3>Email Templates</h3><p style="box-sizing: border-box; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;">The e-mail templates have been completely overhauled to provide a gorgeous design inside your inbox. When recent notifications are sent by e-mail, the design will be more consistent with what is seen in the regular OAE user interface as well as take on the branding configuration of your institution.</p><p style="box-sizing: border-box; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;">In addition to being branded to an institution, the e-mail templates are now completely translated using the standard Crowdin translation approach.</p><p style="box-sizing: border-box; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;">All of this greatly simplifies the maintenance of e-mail templates in Apereo OAE.</p><p style="box-sizing: border-box; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;"><img src="http://i.imgur.com/TLXFsTK.png" alt="Single File Share Email" width="455" height="365" />&nbsp;<img src="http://i.imgur.com/4iMibJP.png?1" alt="2 Comments Email" width="383" height="365" />&nbsp;<img src="http://i.imgur.com/onLe9y4.png?1" alt="2 Emails Some Aggregated" width="236" height="365" /></p><h3>User Management</h3><p style="box-sizing: border-box; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;">OAE Falcon provides the ability for global and tenant administrators to perform a variety of maintenance tasks to help support their users, including:</p><ul class="task-list" style="box-sizing: border-box; padding: 0px 0px 0px 30px; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;"><li style="box-sizing: border-box;">"Become" a user, allowing an administrator to browse the tenant on behalf of the user</li><li style="box-sizing: border-box;">Reset user account passwords</li><li style="box-sizing: border-box;">Update user profile information and visibility</li><li style="box-sizing: border-box;">Create new user accounts</li><li style="box-sizing: border-box;">Search all user accounts in the tenant</li></ul><p><span style="color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif;"><span style="font-size: 15px; line-height: 25.5px;"><img src="http://i.imgur.com/2APndoW.png?1" alt="Administrative User Search" width="397" height="209" />&nbsp;<img src="http://i.imgur.com/6wCNsGJ.png?1" alt="Update User Profile" width="232" height="206" />&nbsp;<img src="http://i.imgur.com/4gl8Vsy.png?1" alt="Administrative Become User" width="396" height="206" /></span></span></p><h3>System Maintenance</h3><p style="box-sizing: border-box; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;">In addition to an enhanced set of tools for managing users, OAE Falcon improves administration for the Global Administrator by providing tools to reindex items in the search index and reprocess content previews based on content and revision filters.</p><h3>Automated Testing</h3><p style="box-sizing: border-box; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;">Support for UI unit tests based on the web browser emulator "PhantomJS" has been added to the automated build process, starting with over 600 unit tests and more to come. These are being added to the existing automated test suite of over 1000 back-end API unit tests, providing even more stability to the release process.</p><h2>Try it out</h2><p style="box-sizing: border-box; margin: 15px 0px; color: #333333; font-family: Helvetica, arial, freesans, clean, sans-serif; font-size: 15px; line-height: 25.5px;">OAE Falcon can be experienced on the project's QA server at&nbsp;<a title="Apere OAE QA Environment" href="http://oae.oae-qa0.oaeproject.org"><span style="color: #4183c4;"><span style="box-sizing: border-box; background-image: initial; background-attachment: initial; background-size: initial; background-origin: initial; background-clip: initial; background-position: initial; background-repeat: initial;">http://oae.oae-qa0.oaeproject.org</span></span></a>.
